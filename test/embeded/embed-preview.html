Notion Embed Test Page

Use this page to verify CSP and iframe behavior.

  1. Start the Next.js app from another terminal: npm run dev.
  2. Serve this file via npx http-server public -p 4000 or any static server.
  3. Visit http://localhost:5000/embed-preview.html and confirm the iframe loads http://localhost:3000.
  4. Verify the Google OAuth buttons escape the iframe and complete sign-in.
